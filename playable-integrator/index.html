🔧 Playable Integrator

小游戏 zip 自动接入 PlayTurbo 平台 SDK

⚠ 此工具需要本地运行，下载后使用（需要 Node.js 环境）

📋 功能说明

Playable Integrator 是一个自动化工具，用于将 HTML5 小游戏接入 PlayTurbo 广告平台 SDK。 工具使用 AI（GLM-5.2）自动分析游戏代码，注入平台 SDK 钩子，并进行 16 项自动化校验， 确保输出的 playable ad 符合平台规范。

🔄 工作流程

  • 上传 zip
    选择打包好的小游戏 zip 文件（需包含 index.html）
  • AI 分析注入
    GLM-5.2 分析游戏引擎（Cocos/Unity/Laya等），自动注入 SDK 钩子代码
  • 确定性后处理
    自动清理 console、重写跳转逻辑、内联资源、替换商店 URL 为 install() 调用
  • 16 项校验
    自动化校验：MRAID 可用性、CTA 按钮钩子、商店 URL、横竖屏方向等
  • 下载结果
    校验通过后下载最终 zip，未完全通过则下载当前最优版本并标注需人工复核项

✅ 16 项自动化校验

  • ✓ MRAID 脚本已加载
  • ✓ mraid.open() 可被调用
  • ✓ install() 函数已定义
  • ✓ CTA 按钮已挂钩 install()
  • ✓ 商店 URL 已替换为 install() 调用
  • ✓ 无外部脚本依赖（全内联）
  • ✓ 无 console.log 残留
  • ✓ window.open 已重定向到 install()
  • ✓ 横竖屏方向正确
  • ✓ 无 MTG 残留代码
  • ✓ 游戏启动事件已触发
  • ✓ 方向变化事件已处理
  • ✓ 资源文件可访问
  • ✓ HTML 语法有效
  • ✓ 文件大小符合平台限制
  • ✓ 无硬编码域名跳转

💻 本地使用

此工具为 Node.js 服务端应用，需要在本地运行。步骤如下：

# 1. 克隆仓库 git clone https://github.com/Tothvillar/aitools.git cd aitools/playable-integrator # 2. 安装依赖 npm install # 3. 配置 API（ARK / Volcengine GLM-5.2） # 编辑 .env 文件，填入 API Token # TOKEN=ark-xxxxxxxx # BASE_URL=https://ark.cn-beijing.volces.com/api/coding # MODEL=glm-5.2 # 4. 启动服务 npm start # 5. 浏览器打开 # http://localhost:3000
注意：需要有效的 ARK API Token（字节跳动火山引擎 GLM-5.2）。 如果没有，可以修改代码使用其他兼容 OpenAI API 格式的 LLM 服务。
